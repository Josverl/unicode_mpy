# Quotation Marks Test - curly and German quotes

This file tests various quotation marks.

## Type: Quotation Marks
- " " English double quotes
- ' ' English single quotes
- „ " German quotes
- « » French guillemets
- 「 」Japanese brackets
- 『 』Japanese double brackets

## Sample:
- "Hello"
- 'World'
- „Hallo"
- «Bonjour»
- 「こんにちは」
